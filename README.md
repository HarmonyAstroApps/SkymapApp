# Skymap: Constellation Finder

Your personal window into the cosmos. Skymap is an iOS astronomy app that helps you identify stars, planets, and constellations in real-time using augmented reality.

## Features

- **Real-time AR Tracking** - Point your device at the sky and instantly identify planets, stars, and satellites
- **Live Sky Map** - Locate Jupiter, Saturn, Mars, Venus, Mercury, Neptune, Uranus, and Pluto with precision
- **Astronomical Data** - Get accurate altitude, azimuth, and orbital positions for all major celestial bodies
- **Constellation Explorer** - Discover constellation stories, mythology, and learn about the Milky Way
- **Solar System Tracker** - Track the Sun, Moon, asteroids, and exoplanets in real-time
- **Educational Content** - Perfect for students, teachers, and amateur astronomers

### Local Development

Simply open `index.html` in a browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```
